o index
WEB-INF에 index.jsp를 만들어서 사용 
> 보안성, controller에서 index를 불러와 사용
> url : http://localhost:8090/pro03/ 
> pro03/ 뒤에 index.jsp가 없이 실행

o web.xml
기존의 코드를 주석처리 or 삭제 후 servlet으로 index가 실행될 수 있도록 구현
> servlet, servlet-mapping 설정
> servlet-name은 동일하도록 설정 
> servlet > index.jsp순으로 작동

o WEB-INF
> view역활

o DB - Maria
> hedisql툴 사용
> 이클립스를 낮은 버전을 사용하기 때문에 db연결을 maria DB로 연결하는 것이 아닌 mysql로 연결하여 사용 
> DB연결은 모두 동일하게 작성하고 maria부분을 mysql로 변경하여 사용

맵핑 : 서로 연결해 주는 것
바인딩 : 데이터를 받는 것
리졸브 : 화면과 주소가 다를 경우  

o 비동기, 동기
> 데이터가 그대로인 상태에서 필요한 부분만 바뀌는 것이 비동기
> 데이터가 모두 바뀌는 것이 동기

o $(id).val() : id 값

o $.ajax({});

o url = 경로, type = 전송방식, ,dataType = 데이터 반환 방식

o 포워드는 데이터를 보낼경우 데이터를 받을 수 없다 
> 받기 위해서는 ajax를 사용해야 한다.

function changeTourNo(){
	var cate = document.frm1.cate.value;
	if(cate == ""){
		alert("분류를 선택하지 않으셨습니다.");
		document.frm1.cate.focus();
		return;
	}
	$.ajax({
		url:"NoLoadCtrl.do"
		type:"post"
		dataType:"json"
		success:functino(data){
			$("#tourno").val(cate+data.no);
		}
	});
}
